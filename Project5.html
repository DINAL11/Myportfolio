Architecture

The application follows a modern microservices architecture with clear separation of concerns.

Frontend Architecture

  • React components with Redux for state management
  • Component-based design for reusability
  • Client-side routing with React Router
  • Responsive design with CSS Grid and Flexbox

Backend Architecture

  • Node.js with Express for RESTful API endpoints
  • MongoDB database with Mongoose ODM
  • JWT-based authentication middleware
  • MVC pattern for organized code structure
